## ⚡ Fastest Way to Start

### Option 1: Docker (Easiest)
```bash
docker-compose up --build
```
Open: http://localhost:3000

### Option 2: Manual
```bash
# Terminal 1 - Backend
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python start_server.py

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev
```
Open: http://localhost:3000
